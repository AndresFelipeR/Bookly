# Bookly

Booking management system built with .NET 9 following Domain-Driven Design and Clean Architecture.

🚧 Work in Progress

## Tech Stack

- .NET 9
- ASP.NET Core
- Entity Framework Core
- SQL Server
- Clean Architecture
- Domain-Driven Design (DDD)
- CQRS (planned)
- MediatR (planned)

## Project Structure

src/
 ├── Bookly.Api
 ├── Bookly.Application
 ├── Bookly.Domain
 ├── Bookly.Infrastructure
